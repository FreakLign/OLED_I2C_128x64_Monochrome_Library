#####OLED Library
This is a library for an 0,96" monochrome organic liquid crystal display.
It has been tested with the following device:
	- 0,96" I2C OLED, Manufacturer: Heltec-Taobao on Arduino UNO

#####Commands
The following commands are implemented:

- initialize
- sendCommand       
- sendData          
- printChar         
- printString       
- printNumber       
- drawBitmap        
- setCursor         
- clearDisplay      
- setBlackBackground
- setWhiteBackground
- setDisplayOff     
- setDisplayOn      
- setPageMode       
- setHorizontalMode 
- setBrightness     
- rotateDisplay180
